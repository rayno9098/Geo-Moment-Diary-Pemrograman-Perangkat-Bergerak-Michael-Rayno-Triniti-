# Geo-Moment Diary

## Deskripsi

Geo-Moment Diary merupakan aplikasi Android yang berfungsi sebagai buku harian digital. Pengguna dapat menyimpan momen penting berupa catatan, foto, tanggal, dan lokasi geografis menggunakan GPS.

---

## Fitur

- CRUD Data Momen
- Room Database
- CameraX
- GPS Otomatis
- Google Maps
- Preview Foto
- RecyclerView
- MVVM Architecture
- Material Design

---

## Teknologi

- Kotlin
- Android Studio
- Room Database
- LiveData
- ViewModel
- RecyclerView
- CameraX
- Glide
- Google Maps SDK
- Fused Location Provider
