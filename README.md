# Geo Moment Diary

Aplikasi Geo Moment Diary merupakan aplikasi mobile berbasis React Native yang digunakan untuk menyimpan momen pengguna beserta foto dan lokasi GPS.

## Fitur

- CRUD Data Momen
- SQLite Database
- Kamera
- GPS Location
- Google Maps
- React Navigation

## Teknologi

- React Native
- Expo SDK 54
- SQLite
- React Navigation
- Expo Image Picker
- Expo Location
- React Native Maps

## Struktur Project

```
src
│
├── components
├── database
├── navigation
├── screens
└── services
```

## Instalasi

```bash
npm install
```

```bash
npx expo start
```
